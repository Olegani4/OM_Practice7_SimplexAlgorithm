1. To run the script open 'RunSimplexMethod.bat' file and enter the file name into the CLI.
2. The input file name should include ".txt". Otherwise, program will not find the file. Also, 
   the input file must be in the same folder as 'RunSimplexMethod.bat' and 'simplexMethodJAR.jar'.
3. When program is finished and no errors occurred, the file 'outputX.txt' or simple 'output.txt'
   will be created in the same folder where executable file is located, or if it is already 
   existing in the same folder the file will be updated.

If program does not work download and install JDK and try again.
Link for downloading JDK: https://www.oracle.com/java/technologies/downloads/#java19

Source code: https://github.com/Olegani4/OM_Practice7_SimplexAlgorithm.git
